## Step 1: Brief Absorption

Synthesize the naming brief from all inputs:

- **Brand personality** (from Identity System)
- **WHY** (the belief the name must carry)
- **Audience** (primary archetype the name must resonate with)
- **Category** (what the name must stand out in)
- **Differentiator** (from Onliness Statement)
- **Constraints** (any domain, trademark, or linguistic requirements)

---

## Step 2: Word Storm

Generate 100+ raw word associations across 5 categories:

| Category | Target | Words |
|----------|--------|-------|
| **Literal** -- What the brand does | 20+ | |
| **Metaphorical** -- What the brand is like | 20+ | |
| **Emotional** -- How the brand makes you feel | 20+ | |
| **Cultural** -- Pop culture, mythology, literature | 20+ | |
| **Linguistic** -- Wordplay, portmanteaus, alliteration | 20+ | |

**Total raw words:** ___/100+

---

## Step 4: SMILE Filter

Score each promising candidate (top 15-20) on SMILE:

| Candidate | S (Suggestive) | M (Memorable) | I (Imagery) | L (Legs) | E (Emotional) | Total /25 |
|-----------|---------------|---------------|-------------|----------|---------------|-----------|
| | /5 | /5 | /5 | /5 | /5 | |

**Threshold:** Keep candidates scoring >= 18/25

---

## Step 5: SCRATCH Filter

Check survivors for red flags:

| Candidate | S (Spelling) | C (Copycat) | R (Restrictive) | A (Annoying) | T (Tame) | C (Curse) | H (Hard to say) | Clean? |
|-----------|-------------|-------------|-----------------|-------------|----------|----------|-----------------|--------|
| | | | | | | | | |

Any flag = elimination unless the name scores 23+ on SMILE.

---

## Step 6: Shortlist & Domain Check

### Top 3 Recommendations

For each of the top 3:

**Name Candidate #N: [NAME]**
- **SMILE Score:** __/25
- **SCRATCH Flags:** None / [list]
- **Why this name works:** Strategic rationale connecting to WHY and Onliness
- **Domain options:** .com availability, alternative TLDs, creative domain hacks
- **Verbal test:** How it sounds when spoken aloud
- **Creative potential:** Tagline, campaign, sub-brand extension ideas
