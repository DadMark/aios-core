## Step 1: VOICE Framework

### V -- Vernacular

| Dimension | Definition |
|-----------|-----------|
| **Always words** (10) | |
| **Never words** (10) | |
| **Vocabulary level** | 7th grade / High school / College / PhD |
| **Jargon policy** | Plain language / Industry terms OK / Technical |
| **Sentence length** | Short (< 15 words) / Medium / Long / Varied |
| **Contractions** | Always / Sometimes / Never |

### O -- Originality

| Dimension | Definition |
|-----------|-----------|
| **Logo-off test** | Could someone identify the brand with logo covered? How? |
| **Verbal tics** | Unique phrases, recurring patterns |
| **Sentence structure** | Signature patterns (fragments? questions? lists?) |
| **Humor style** | Witty / Dry / Playful / No humor / Situational |

### I -- Identity

| Dimension | Definition |
|-----------|-----------|
| **If the brand were a person** | How would they speak? |
| **5 personality adjectives** | 1. 2. 3. 4. 5. |
| **Famous voice equivalent** | "Speaks like ___ meets ___" |
| **Energy level** | Calm / Warm / Energetic / Intense |

### C -- Consistency

| Dimension | Definition |
|-----------|-----------|
| **Non-negotiable rules** | 3-5 rules that never bend |
| **Quality check** | How to review copy for voice compliance |
| **Training** | How new writers learn the voice |
| **Do/Don't examples** | 3 pairs showing right/wrong |

### E -- Empathy

| Dimension | Definition |
|-----------|-----------|
| **Audience worries** | Top 3 concerns the audience has |
| **Pain acknowledgment** | How the brand validates frustration |
| **Emotional state** | What the reader feels when encountering the brand |
| **Empathy signals** | Specific phrases/approaches that show understanding |
