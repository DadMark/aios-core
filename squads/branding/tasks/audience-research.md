## Step 1: Audience Archetype Mapping

Map the brand's audience to the 5 archetypes. Most brands have a primary and secondary archetype.

| Archetype | Description | Present? | Brand Approach |
|-----------|-------------|----------|----------------|
| **Maven** | Information specialist, researches deeply | | Depth, specs, transparency |
| **Connector** | Social hub, links people and ideas | | Easy sharing, social currency |
| **Salesman** | Charismatic persuader, drives word-of-mouth | | Story worth telling |
| **Outlier** | Early adopter, seeks unconventional | | Exclusivity, innovation |
| **Mainstream** | Follows proven social proof | | Numbers, testimonials |

**Primary Archetype:** _______________
**Secondary Archetype:** _______________

For each active archetype, develop:
- Demographic snapshot
- Psychographic profile (values, beliefs, lifestyle)
- Media consumption habits
- Brand decision journey
- Key trigger points

## Step 2: Tipping Point Analysis

### Law of the Few

| Role | In This Market | Strategy |
|------|---------------|----------|
| Connectors | Who are the network hubs? | |
| Mavens | Who are the knowledge brokers? | |
| Salesmen | Who are the natural persuaders? | |

### Stickiness Factor

| Test | Result |
|------|--------|
| Can someone repeat the brand promise after hearing it once? | |
| Does it pass the Sesame Street test (memorable AND actionable)? | |
| Does it pass the cocktail party test (shareable in conversation)? | |

### Power of Context

| Question | Answer |
|----------|--------|
| Where does the audience first encounter the brand? | |
| What cultural moment is the brand riding? | |
| What's the "broken windows" equivalent in this category? | |

## Step 3: Blink Perception Audit

What snap judgment does the brand create?

| Layer | Question | Initial Assessment |
|-------|----------|-------------------|
| Visual | What does someone feel in 2 seconds looking at the brand? | |
| Verbal | What does someone feel hearing the brand name? | |
| Experiential | What does the first 10-second interaction communicate? | |
| Social | What do people assume about someone who uses this brand? | |

---

## Step 4: Cultural Context Map

| Dimension | Analysis |
|-----------|----------|
| **Zeitgeist** | What cultural wave is the brand riding? |
| **Counter-culture** | What does the brand reject? |
| **Tribal identity** | What group identity does the brand reinforce? |
| **Status signals** | What does owning this brand say about you? |
| **Narrative fit** | How does the brand fit into the customer's life story? |

---

## Step 5: Insight Synthesis

Compile the Consumer Insight Report with:

1. **Primary Audience Archetype** -- detailed profile with approach strategy
2. **Secondary Audience Archetype** -- profile and how it differs
3. **Tipping Point Playbook** -- specific strategies for Few, Stickiness, Context
4. **Perception Risks** -- what the Blink audit reveals that needs attention
5. **Cultural Opportunity** -- the zeitgeist alignment that the brand should leverage
6. **Key Insight** -- the single most important thing discovered (tell it as a story)
